# GitHub UI Guide

GitコマンドとGitHub UIの関係を視覚的に学べるインタラクティブなシミュレーター。

## Features

- コマンド入力 → GitHub UI が変化
- `git add`, `git commit`, `git push` などを体験
- スプレッドシートでデータ管理

## Setup

1. `src/js/config.example.js` をコピーして `config.local.js` を作成
2. GAS Web App URL を設定
3. `src/index.html` をブラウザで開く

## Tech Stack

- HTML/CSS/JavaScript (Vanilla)
- Google Apps Script
- Google Spreadsheet
